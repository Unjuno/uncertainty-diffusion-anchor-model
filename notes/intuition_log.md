# Intuition Log

## Intuition 1: Restarting is not meaningless

If a count fails once, it does not follow that all later counting is meaningless.

Only the lost interval becomes uncertain.

## Intuition 2: Uncertainty does not merely stay fixed

After losing an anchor, uncertainty can grow over time.

This is the central correction from a static uncertainty model to an uncertainty diffusion model.

## Intuition 3: Action can be measurement

A small action can serve as a measurement of the current state.

Examples:

- one problem measures skill;
- one message measures social state;
- one symptom log measures health state;
- one project inspection measures work state.

## Intuition 4: Positive high-frequency action can be rational

If uncertainty diffuses and small informative actions are cheap, repeated re-anchoring can be advantageous.

This is not arbitrary hyperactivity.

It is high-frequency low-cost re-anchoring.

## Intuition 5: The first goal after collapse is not success

After losing the model of the current state, the first goal is to re-establish contact with reality.

In other words:

> First re-anchor. Then optimize.
