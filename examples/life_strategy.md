# Example: Life Strategy After Failure

## Situation

After a major failure, the agent feels that their life, career, or future has ended.

UDAM treats this not as a confirmed terminal state, but as a possible collapse of the agent's internal model.

## Lost anchor

The lost anchor is self-state:

- ability;
- resources;
- social standing;
- next viable action;
- remaining options.

## Uncertainty diffusion

If the agent does nothing, uncertainty may grow about:

- whether skills remain usable;
- whether opportunities remain;
- whether relationships are repairable;
- whether action is still possible.

```text
P_{self,t+Δt} = P_{self,t} + Q_self Δt
```

## Re-anchoring action

Choose one minimal action that contacts reality.

Examples:

- write one sentence;
- solve one problem;
- send one message;
- apply to one opportunity;
- clean one visible area;
- work for five minutes.

## Informational value

The action tests whether the agent is truly unable to act, or merely uncertain about their state.

```text
I(a) > 0
```

## Intervention value

The action may also improve the state slightly.

```text
B(a) >= 0
```

## Failure case

This model does not say that all action solves all problems.

If the agent is in immediate danger or severe crisis, safety and professional support take priority over self-experimentation.

## Practical claim

After model collapse, the first goal is not success.

The first goal is re-anchoring.
